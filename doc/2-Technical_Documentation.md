# Technical Documentation

## User Stories

## User Stories Table

| Category            | User Role       | User Story Description                                                                 | Priority   |
|---------------------|-----------------|-----------------------------------------------------------------------------------------|------------|
| **Discovery**       | Normal User     | As a normal user, I want to search and find events around me or in a specific city/village, so I can explore activities in my desired location | Must Have |
| **Discovery**       | Normal User     | As a normal user, I want the app to detect my location automatically, so that I can see events happening near me without manually entering my location | Must Have |
| **Discovery**       | Normal User     | As a normal user, I want to filter events by category, so that I can easily find the types of events that interest me                             | Must Have |
| **Discovery**       | Normal User     | As a normal user, I want to see a list of events near my current location, so that I can quickly browse and choose events to attend                | Must Have |
| **Event Details**   | Normal User     | As a normal user, I want to view detailed information about an event, so that I can decide whether to attend and get all the necessary details      | Must Have |
| **Event Management**| Event Creator   | As a creator, I want to be able to create, edit, and manage my events, so that I can keep my event information up-to-date and accurate             | Must Have |
| **Profile Management** | Normal User | As a user, I want to create an account, so that I can save my favorite events and receive notifications                                          | Must Have |
| **Profile Management** | Normal User | As a user, I want to add events to my favorites, so that I can easily access them later                                                          | Must Have |
| **Profile Management** | Normal User | As a user, I want to receive notifications for my favorite events, so that I don't miss any updates                                               | Must Have |
| **Profile Management** | Normal User | As a user, I want to manage my profile, so that I can update my information and preferences                                                       | Must Have |
| **UI/UX**           | Normal User     | As a user, I want the app to be optimized for my iPhone, so that I have a seamless and enjoyable experience using the app                         | Must Have |
| **UI/UX**           | Normal User     | As a user, I want the app to have a simple and intuitive interface, so that I can easily navigate and find what I need                          | Must Have |
| **UI/UX**           | Normal User     | As a user, I want to switch between dark and light modes, so that I can use the app comfortably in different lighting conditions                  | Must Have |
| **Backend**         | Developer       | As a developer, I want to use Firebase for the backend, so that I can quickly and easily set up and manage the app's data and authentication     | Must Have |
| **Testing**         | QA Tester       | As a QA tester, I want to ensure the app is thoroughly tested, so that users have a reliable and bug-free experience                             | Must Have |
| **Marketing**       | Visitor         | As a visitor, I want to see a landing page that introduces the app and its features, so that I can understand what the app offers and decide to download it | Must Have |
| **Social Features** | Normal User     | As a user, I want to be able to type in a city name to search for events, so that I can find events in specific locations even if I'm not currently there | Should Have |
| **Social Features** | Normal User     | As a user, I want to filter events by category, so that I can easily find the types of events that interest me                                    | Should Have |
| **Social Features** | Normal User     | As a user, I want to partager des événements sur les réseaux sociaux, afin d'inviter des amis à me rejoindre. | Should Have |
| **Verification**    | Event Creator   | As a creator, I want to verify my account with SIREN/SIRET, so that users can trust my events                                                     | Should Have |
| **Reviews**         | Normal User     | As a user, I want to see the avis et les notes des événements, afin de prendre des décisions éclairées sur ma participation.                      | Should Have |
| **Personalization** | Normal User     | As a user, I want to receive recommendations d'événements basées sur mes préférences, afin de découvrir de nouvelles activités.                    | Should Have |
| **Advanced Search** | Normal User     | As a user, I want to see une vue calendrier des événements, afin de pouvoir planifier mon emploi du temps.                                      | Could Have |
| **Advanced Search** | Normal User     | As a user, I want to receive des notifications personnalisées basées sur mes préférences, afin de rester informé des événements pertinents.       | Could Have |
| **Analytics**       | Event Creator   | As a creator, I want to see les analyses de mes événements, afin de comprendre leur popularité et leur portée.                                    | Could Have |
| **Communication**   | Normal User     | As a user, I want to discuter avec les organisateurs d'événements, afin de poser des questions et obtenir plus d'informations.                    | Could Have |
| **Trending Events** | Normal User     | As a user, I want to voir une liste des événements tendance, afin de rester à jour avec les activités populaires.                                 | Could Have |
| **Purchasing**      | Normal User     | As a user, I understand that I will not be able to purchase tickets directly within the app, so I will need to use external methods for ticket purchasing | Won't Have |
| **Social Media Integration** | Normal User | As a user, I understand that social media integration for sharing events or logging in is not available in this version of the app.             | Won't Have |
| **Language Change** | Normal User     | As a user, I understand that the app will be available in a single language, and changing the language is not an option.                          | Won't Have |
| **Reviews**         | Normal User     | As a user, I understand that user and event reviews are not available in this version but may be included in future updates.                     | Won't Have |
| **Advanced Search** | Normal User     | As a user, I understand that advanced search filters like date, price range, and event popularity are not available in this version but may be included in future updates. | Won't Have |
| **Reminders**       | Normal User     | As a user, I understand that event reminders and push notifications for upcoming events are not available in this version.                       | Won't Have |
| **Ticketing**       | Normal User     | As a user, I understand that direct ticket purchasing through the app is not available in this version.                                          | Won't Have |
| **Recommendations** | Normal User     | As a user, I understand that personalized recommendations based on my preferences and past behavior are not available in this version.           | Won't Have |
| **Live Events**     | Normal User     | As a user, I understand that live streaming of digital events is not available in this version of the app.                                       | Won't Have |
